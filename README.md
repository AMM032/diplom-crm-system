## Дипломный проект: CRM-система учета клиентов и заказов со встроенной справочной системой.

Описание проекта
Автоматизированная система учета клиентов и заказов (CRM) со встроенной справочной системой и поддержкой мобильных устройств. Проект разработан в рамках выпускной квалификационной работы для автоматизации процессов подачи заявок на транспортное обслуживание.

Ключевые функции:
- Учет клиентов и ведение реестра заказов на автобусы.
- Встроенная справочная система (FAQ, инструкции, подсказки).
- Адаптивный веб-интерфейс для ПК и мобильных устройств.
- Ролевая модель доступа (Менеджер, Администратор).
